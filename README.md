# Capital Bikeshare Before & During COVID-19
15 Jul 2020

## Description

The purpose of this project is to analyze capital bikeshare (cb) usage and trends around Washington, DC before and during the COVID-19 ('coronavirus') pandemic. Data is provided by Capital Bikeshare [here](https://www.capitalbikeshare.com/system-data).

What is Capital Bikeshare? From the [site](https://www.capitalbikeshare.com/how-it-works):
>Capital Bikeshare is metro DC's bikeshare service, with 4,500 bikes and 500+ stations across 7 jurisdictions: Washington, DC.; Arlington, VA; Alexandria, VA; Montgomery, MD; Prince George's County, MD; Fairfax County, VA; and the City of Falls Church, VA.
